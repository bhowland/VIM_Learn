# VIM Start Guide

## Command | Count | Motion

### Command

d/x - delete \
u - undo \
r - redo \
y - yank(Copy) \
p - paste

### Motion

J - Down \
K - Up \
L - Right

+ W - Word ->

H - Left

+ B - Word <-

### insert mode

i - insert \
a - insert after

### Visual Mode (Highlight mode)

v - visual mode \
V - line visual mode
